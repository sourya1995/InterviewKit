# 🚀 The Ultimate Interview Kit

A comprehensive collection of resources for Software Engineering interviews, covering everything from Data Structures and Algorithms to System Design and Frontend Development.

---

## 📁 Repository Structure

### 🧠 [DSA (Data Structures & Algorithms)](./DSA/)
Focuses on coding patterns, problem trackers, and LeetCode strategies.
*   **6-Month DSA Problem Tracker**: A structured 1000-problem plan.
*   **coding_patterns_ref.md**: Essential patterns (Sliding Window, Two Pointers, etc.).
*   **LegendLeetcode.pdf & ProblemRoadmap.pdf**: Strategy guides for cracking LeetCode.

### 🏗️ [System Design](./System-Design/)
High-level architectural concepts and real-world system diagrams.
*   **API Design Interview Study Sheet**: Cheatsheet for designing robust APIs.
*   **BOTECalc.md**: "Back Of The Envelope" calculation guide.
*   **Case Studies**: Diagrams for DoorDash, Sportalytics, and Order Fulfilment workflows.

### 🌐 [Distributed Systems](./Distributed-Systems/)
Deep dives into large-scale systems and patterns.
*   **Patterns of Distributed Systems (Unmesh Joshi)**: The definitive guide to distributed patterns.
*   **distributed_systems_patterns_by_system.pdf**: Patterns categorized by system type.

### 📐 [LLD (Low-Level Design)](./LLD/)
Object-Oriented Design and practice problems.
*   **lld-cheatsheet.html**: Key LLD principles and patterns.
*   **Multi-Tier C# Practice**: Progressive LLD challenges (Tier 1 to 4).

### 💻 [Languages & Frameworks](./Languages/)
Deep dives into specific languages and their ecosystems.
*   **Java**: Core Java, Spring Boot, Microservices, and Java design patterns.
*   **CSharp**: .NET prep, Kafka integration, and MongoDB.
*   **Python**: AWS interview prep and Machine Learning basics.
*   **JS & SQL**: JavaScript examples and SQL query preparation.

### 🎨 [Frontend](./Frontend/)
Modern web development and performance.
*   **Advanced React**: Deep dives into performance and patterns by Nadia Makarevich.
*   **Web Performance Fundamentals**: Optimizing frontend applications.
*   **React Cheatsheet**: Quick reference for hooks and components.

### 👔 [Career & General](./Career-and-General/)
General engineering wisdom and interview playbooks.
*   **Software Engineer's Guidebook (Gergely Orosz)**: Career navigation from Senior to Staff level.
*   **Interview Practice Playbook**: Strategic advice for the interview process.
*   **Security & Concurrency**: Specialized guides for niche technical topics.

---

## 🚦 How to Use This Kit
1.  **If you're a beginner**: Start with **DSA** and **Languages**.
2.  **Preparing for System Design**: Focus on the **System-Design** and **Distributed-Systems** folders.
3.  **For Senior Roles**: Deep dive into the **Software Engineer's Guidebook** in **Career-and-General**.

Happy Coding! 🚀
